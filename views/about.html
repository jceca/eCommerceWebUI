Sobre la página Tecnologías

La tecnología que se ha escogido para la realización de la página es la de un framework desarrollado por Google, AngularJS.


Fig. 1 - AngularJS (www.angularjs.org).

AngularJS es un framework de JavaScript de código abierto, mantenido por Google, que ayuda con la gestión de lo que se conoce como aplicaciones de una sola página. Está basado en el MVC, Modelo Vista-Controlador en un intento de hacer el desarrollo y testing más fáciles. Todos los frameworks tienen una serie de características que los diferencian de otros. Sabiendo como explotar correctamente estas características para crear potentes aplicaciones, AngularJS se caracteriza por:
  • Comprensión del funcionamiento interno de los templates
  • Data-binding bidireccional para fortalecer la relacion entre las vistas y los datos que controlan.
  • Dependency Injection, AngularJS inyecta las dependencias sin necesidad de perder el tiempo referenciandolas.
